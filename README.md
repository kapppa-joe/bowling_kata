# Bowling Kata

Pair programming on 2022/02/10 with @CatherineDeas


Link of the Bowling kata:
https://learn.madetech.com/katas/bowling/

## To clone this repo:
run `git clone https://github.com/kapppa-joe/bowling_kata.git`

## To set up the gems:
run `bundle install`

## To run the tests:
run `bundle exec rspec`
